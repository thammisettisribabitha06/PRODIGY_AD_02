# PRODIGY_AD_02 – To-Do List App

## Description

This project is a simple **To-Do List Android application** developed as part of the **Prodigy InfoTech Android App Development Internship (Task-02)**. The app helps users manage their daily tasks by allowing them to add, edit, and delete tasks.

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Simple and easy-to-use interface

## Technologies Used

* Java
* XML
* Android Studio

## How It Works

1. Enter a task in the input field.
2. Click **Add Task** to add it to the list.
3. Tap the task text to edit the task.
4. Click **Delete** to remove the task from the list.
